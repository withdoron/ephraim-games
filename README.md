# Ephraim Games 🎮

Games designed and built by Ephraim.

## Games

### 🧗 GRIP

A rock climbing arcade game. Choose a monster, climb the wall, dodge dinosaurs, send the route.

### ✈️ Ringstorm Racers

A flying racing game. Race through rings, collect powerups, battle with homing missiles. 1-2 players.

## Running Locally

```bash
# GRIP
cd grip && npm install && npm run dev

# Ringstorm Racers
cd ringstorm && npm install && npm run dev
```
