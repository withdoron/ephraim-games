# Godot Migration Plan

## When to Migrate

* Game design is locked (controls feel right, modes are complete, fun factor confirmed)
* Prototype has been playtested by Ephraim and friends
* Ready for real 3D graphics and online multiplayer

## Tech Stack

* **Godot 4.x** (open source game engine)
* **GDScript** (Python-like, beginner-friendly)
* **Nakama** (open source multiplayer server) for online play

## Migration Order

1. **GRIP first** (simpler, 2D)
2. **Ringstorm Racers second** (3D, more complex)

## Publishing Path

* **itch.io** (free, web + desktop builds)
* **locallane.app/games** (community integration)
* **Steam** (long-term, if games grow)
